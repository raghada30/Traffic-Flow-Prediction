# Traffic Flow Prediction 

This project focuses on predicting traffic flow using machine learning techniques
based on real-world traffic data.

#Project Overview
The goal of this project is to analyze traffic data and build predictive models
that can estimate traffic flow and related metrics. The project applies data
preprocessing, feature engineering, and regression models to achieve accurate
predictions.

#Tools & Technologies
- Python
- Google Colab
- Pandas & NumPy
- Scikit-learn
- Matplotlib / Seaborn

#Machine Learning Models
- Support Vector Regression (SVR)
- Random Forest Regressor
- Hyperparameter tuning using GridSearchCV
- Cross-Validation for model evaluation

#Dataset
The dataset is based on real-world traffic data collected from traffic monitoring
stations and includes features such as:
- Date & Time
- Traffic Flow
- Speed
- Occupancy

#Evaluation Metrics
- RMSE (Root Mean Squared Error)
- MAE (Mean Absolute Error)
- R² Score

#How to Run
1. Open the notebook using Google Colab
2. Upload the dataset files
3. Run all cells sequentially
